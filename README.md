# Chomikuj Link Generator

## Opis projektu
Aplikacja umożliwia generowanie linków do pobrania plików z Chomikuj.pl.

- **Frontend**: Hostowany na GitHub Pages.
- **Backend**: Hostowany na Render, obsługuje generowanie linków poprzez API.

---

## Struktura projektu

## Jak uruchomić
### Frontend
1. Frontend jest dostępny pod adresem: [GitHub Pages](https://garfieldowce.github.io/CLG/).

### Backend
1. Backend jest hostowany na Render pod adresem: `https://twoj-backend.onrender.com`.

---

## Technologie
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express, Puppeteer
